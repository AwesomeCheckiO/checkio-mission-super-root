Square roots, cube roots, 4th roots... It's too boring for Nicola. He needs the super root! And he will find it.

The super root of a number a is the number b, such that bb = a.

The result should be accurate so that bb ≈ a±0.001. Or a - 0.001 < bb < a + 0.001.

Input: A number as an integer.

Output: The super root as a float or an integer.

Example: 

super_root(4) == 2
super_root(27) == 3
super_root(81) == 3.504339593597054

How it is used: This concept can be useful for the cryptography. And you will look how work your calculator then calculate roots.
